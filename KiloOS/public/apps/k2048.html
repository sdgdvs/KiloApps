How to Play

Controls: Arrow keys, WASD, or swipe to move tiles.

Hotkeys:
U / Ctrl+Z: Undo
P: Auto-Play
O: Toggle Obstacles
F: Toggle Fibonacci Mode
T: Change Theme
I: Statistics
M: Time Attack
3, 4, 5, 6: Change Grid Size

Fibonacci Mode: Merge adjacent Fibonacci numbers (1+1=2, 1+2=3, 2+3=5, etc.).

Obstacles: Unmergeable blocks drop occasionally.

Time Attack: You have 60 seconds. Game over when time runs out.
