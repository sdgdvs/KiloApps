How to Play

Controls: Arrow keys, WASD, or swipe to move tiles.

Hotkeys:
U / Ctrl+Z: Undo
E: Shuffle Powerup
Q: Hammer Powerup
X: Rotate Powerup
V: Upgrade Powerup
P: Auto-Play
O: Toggle Obstacles
B: Toggle Bombs
F: Toggle Fibonacci Mode
T: Change Theme
I: Statistics
M: Time Attack
3, 4, 5, 6: Change Grid Size

Fibonacci/Threes Mode: Merge Fib numbers or 1+2=3, 3+3=6.

Obstacles/Bombs: Unmergeable blocks drop occasionally. Bombs destroy any tile.

Time Attack: You have 60 seconds. Game over when time runs out.
